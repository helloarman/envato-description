# CodeCanyon HTML Description Generator

A free, open-source tool designed to help Envato authors create professional, compliant HTML item descriptions for CodeCanyon and ThemeForest without writing code.

**Live Demo:** [https://helloarman.github.io/envato-description](https://helloarman.github.io/envato-description)

## üöÄ Features

- **Block-Based Editor**: Build your description using modular blocks like Titles, Text, Images, Tables, and Lists.
- **Envato Compliant**: Generates strict, clean HTML with no inline CSS, ensuring your item isn't soft-rejected during review.
- **Real-Time Preview**: See exactly how your description will look on the marketplace as you edit.
- **Structure View**: A dedicated sidebar to reorder blocks via drag-and-drop or rename sections for better organization.
- **Specialized Blocks**:
  - **Credentials**: Pre-formatted table for sharing demo login details.
  - **Changelog**: Standardized version history formatting.
  - **Image Rows**: Automatically creates responsive table layouts for side-by-side images.
- **Auto-Save**: Your work is automatically saved to your browser's local storage so you never lose progress.
- **HTML Source Editor**: Direct access to the raw HTML code with bidirectional updating (edit code ‚Üí update view).

## üõ†Ô∏è How to Use

1. **Add Blocks**: Use the sidebar to add elements like Title, Details, Image, or Credentials.
2. **Edit Content**: Fill in the fields in the visual editor. Use the toolbar buttons for bolding text or adding links.
3. **Organize**: Switch to the Structure tab to drag-and-drop blocks into the perfect order.
4. **Preview**: Toggle the Preview mode (Eye icon) to check the final look.
5. **Export**: Go to the HTML Source tab and click Copy HTML.
6. **Publish**: Paste the code directly into the description field of your CodeCanyon/ThemeForest item submission.

## üíª Installation & Development

This project is built as a **Single File Application**. It requires no build steps, Node.js, or complex server setup. It runs entirely in the browser using React and Tailwind CSS via CDN.

### To Run Locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/helloarman/envato-description.git
   ```

2. Open `index.html` in any modern web browser.

That's it!

### Technologies Used

- **React 18** (via UMD)
- **Tailwind CSS** (via CDN)
- **Babel** (Standalone for in-browser JSX compilation)
- **Lucide Icons**
